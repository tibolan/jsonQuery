jsonQuery by tibolan

A json selector, inspired by xpath with javascript adjustment.
Simple & Customizable. 

jsonQuery.parse("store.products[family = 'geek' && price < 50]", JSON);
jsonQuery.parse("store.products[myOwnFilter()]", JSON);

Features

  • Simple pointer to a property

    • jsonQuery.parse("path.to.the.property", JSON);
    • jsonQuery.parse("path.to.the[property]", JSON);

  • Filtering Array

    • jsonQuery.parse("path.to.an.array", JSON);
    • jsonQuery.parse("path.to.an.array[1]", JSON); // only if Array
    • jsonQuery.parse("path.to.an.array[position() = last()]", JSON);
    • jsonQuery.parse("path.to.an.array.last()", JSON);

  • Filtering Hash

    • jsonQuery.parse("path.to.an.hash", JSON);
    • jsonQuery.parse("path.to.an.hash[id]", JSON);
    • jsonQuery.parse("path.to.an.hash[age > 18]", JSON);

  • Create your own filtering method.

    • jsonQuery.parse("object[myFilter()]", JSON);
    • jsonQuery.parse("array[myFilter(property1, property2, 'whatever_u_need')]", JSON);
